============
Installation
============

At the command line::

    $ pip install ynabinterfaceslib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv ynabinterfaceslib
    $ pip install ynabinterfaceslib

Or, if you are using pipenv::

    $ pipenv install ynabinterfaceslib

Or, if you are using pipx::

    $ pipx install ynabinterfaceslib
